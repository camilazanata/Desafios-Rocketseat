HTML Glossary

<html></html>
This tag is always in the beggining and in the ending of the web page. It bounders where the code start and end.

<head></head>
This tag is used to determine the beggining and ending of a HTML document header.

<meta></meta>
This tag informs basics data about the document, such as the author and the creation date.

<title></title>
Defines the HTML document title. This name is shown on the top of the browser window when the page is open.

<body></body>
This tag indicates to the browser the HTML document body, that is, the main part of a web page. This tag is, usually, followed by other elements.
